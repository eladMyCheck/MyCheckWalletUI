CheckoutDelegate

public protocol CheckoutDelegate

The parent of the MCCheckoutViewController must adopt this protocol and implement its methods in order to be able to resize the view when needed. The View will not automatically resize since it might be used in a few different ways (e.g constraints might be broken), so it is your responsibility to respond to the delegate and resize the view appropriately.

  • checkoutViewShouldResizeHeight(_:animationDuration:)

    Called when the height is changed - parameter newHight: The new height of the CheckoutView/CheckoutTableViewCell - parameter animationDuration: The duration the animation will take. The animation will start directly after this call is pressed. You should resize the view imidiatly and use the same animation duration in order for the animation to look good
